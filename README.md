# Estudio de Mosaicos 10x15

Aplicativo estatico para montar fotos em folhas 10x15 cm para impressao.

Esta versao publicada no GitHub Pages nao inclui fotos pessoais embutidas. As imagens sao carregadas pelo usuario diretamente no navegador e ficam salvas localmente no proprio dispositivo via IndexedDB.

Site: https://deivs.github.io/mosaico/

## Uso

Abra o site, envie as fotos, escolha o tamanho do recorte e exporte JPGs ou PDF para imprimir.

## Observacao de privacidade

O arquivo local `estudio-mosaicos.html` do projeto original continha fotos pessoais embutidas em base64. Ele nao foi publicado neste repositorio remoto.
