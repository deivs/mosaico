# Estudio de Mosaicos 10x15

Aplicativo estatico para montar fotos em folhas 10x15 cm para impressao.

Esta publicacao no GitHub Pages inclui a base de imagens embutida no `index.html`. O site roda direto no navegador e as imagens iniciais ja aparecem ao abrir.

Site: https://deivs.github.io/mosaico/

## Uso

Abra o site, escolha a montagem e exporte JPGs ou PDF para imprimir.

## Observacao

Como as imagens estao embutidas no HTML, qualquer pessoa com acesso ao repositorio ou ao site publico pode visualizar esse conteudo.
